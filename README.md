* 그리드: 정렬을 시작하면 레이아웃이 일종의 구조를 형성하기 시작하는것
* 정말 좋은 디자인은 같은 크기로 나누어진 그리드
* 그리드는 명료한 느낌을 준다
* 반복되는 그리드 덕분에 무언가가 예측 가능하고 친숙해 보일때 바움ㄴ자는 웹 사이트를 더 신뢰하게 될것이며 이 웹사이트에서 시간을 낭비하지 않으리라는 점을 알게된다
* 그리드의 규칙은 유연하다
* 그리드는 전혀 사용하지 않을수도 있다
* 그리드를 사용하는것은 멋진 디자인 습관을 주입하 수 있는 아주 놀라운 방법
* 인쇄디자인에서는 중앙 정렬을 거의 사용하지 않음
* 웹사이트는 작은 스마트폰에서 큰 tv화면에 이르기 까지 화면크기가 다르기때문에 반응성이 좋아짐 -> 정렬과 그리드 활용

* 디자너가 알아야 할 디자인 방법
  1. 대비: 색상대비, 크기대비, 무게감대비
  2. 여백: 
     1. 웹사이트의 주요 목적이 검색이라면 그것에 집중할 수있게 해야한다. 
     2. 여백이 많아서 불평하는 사용자는 없지만 복잡한 페이지를 싫어하는 사용자는 많다. 
     3. 여백이 많은 디자인은 과감하다. 
     4. 좋은 디자인은 최소한의 디자인이다 여백을 두려워하지 말라
     5. 여백의 예외: 서로 관련되거나 맥락이 같은 요소는 가깝게 붙여야한다
  3. 반복: 디자인은 인간의 심리와 연관되어 있다. 디자인에 적욧ㅇ되는 물리법칙은 없다 완벽이 인간 중심적, 모든 인간은 디자이너이고 반복과 여백  대비등의 디자인 요소를 원하는건 인간
  4. 일관성:
     1. 좋은 디자인을 구분하는 가장쉬운 방법은 일관성, ex) 웹사이트의 레이아웃, 타이포그래피, 색상의 일관성, 도형의 일관성
     2. 인간은 패턴을 벗어난 것을 쉽게알아본다. 정확히 몰라도 쉽게 인지가능, 
     3. 일관성 있는 나쁜 디자인이 일관성 없는 좋은 디자인보다 낫다
     4. 부드러운 도형을 사용하다 각진 도형을 사용하면 일관성이 깨져 어색하다
     5. 사진의 촬영을 다르게하거나 배경이 달라도 일관성이 깨질 수 있다.
     6. 아이콘을 고를때는 패키지 단위로 하는게 좋음
  5. 겹치기:
     1. 겹치기는 그리드와 반대개념, 겹치기는 의도적으로 배열을 깨서 구성을 더 재미있게 한다
     2. 배열을 의도적으로 개는것은 괜찮다
  6. 긴장감:
     1. 기울이기
     2. 균형을깨거나 일부를 가리기 ex) 익스트림크롭
